# cooked - a minimalistic recipe web-viewer

**cooked** is a web-based recipe catalog and viewer focused on improving the user experience in reading and following online recipes.

## Demo

A short demo video is available at https://youtu.be/sqkhWE_z2vU.

## Deployment

_Want to try the app without running the project locally?_

We've deployed it for your convenience to the web using [vercel](https://vercel.co).

The app is available at https://s357-cooked.vercel.app/.

## Getting started

### Prerequisite

To run **cooked** locally, you'll need to install either [bun](https://bun.com/) or [node](https://nodejs.org/en).

### Installing dependencies

Run either `bun i` or `npm i` from the root to install the project dependencies.

### Running the dev server

Run the dev script with `bun run dev` or `npm run dev`. The Vite dev server can then be found at http://localhost:5173/
